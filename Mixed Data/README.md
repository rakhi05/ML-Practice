# 🔀 Mixed Data Feature Engineering with Pandas

A beginner-friendly Jupyter Notebook that demonstrates how to handle **mixed data columns** (containing both text and numbers) using **Pandas**. The notebook uses the **Titanic dataset** to extract meaningful features from columns such as **Cabin**, **Ticket**, and **Number** for machine learning preprocessing.

---

## 📌 Overview

Real-world datasets often contain columns with a combination of **alphabetic and numeric values**. Before training a machine learning model, these values need to be separated into meaningful features.

This notebook shows how to:
- Identify mixed data columns
- Separate numerical and categorical information
- Create new features for analysis and modeling

---

## 📚 Topics Covered

### 🔢 Number Column
- Explore unique values
- Analyze value distribution
- Extract numerical values
- Separate categorical values

### 🚢 Cabin Column
- Extract cabin number
- Extract cabin category (deck letter)
- Handle missing values
- Visualize cabin categories

### 🎫 Ticket Column
- Extract ticket number
- Extract ticket prefix/category
- Convert extracted values into appropriate data types
