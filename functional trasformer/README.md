# FunTransformer - Feature Engineering with Titanic Dataset

## 📌 Overview
This project demonstrates how to use **FunctionTransformer (FunTransformer)** for feature engineering and data preprocessing on the Titanic dataset.

The notebook explores different transformations, handles missing values, and visualizes data distributions before applying transformations.

## 🚀 Features
- Load and explore the Titanic dataset
- Handle missing values
- Apply FunctionTransformer
- Perform log transformation
- Analyze Age and Fare distributions
- Generate Histogram and QQ Plots
- Prepare data for Machine Learning models

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Scikit-learn

## 📂 Dataset
- Titanic Dataset (`train.csv`)

## 📊 Concepts Covered
- Data Cleaning
- Missing Value Imputation
- Feature Transformation
- Normal Distribution
- QQ Plot
- Histogram
- FunctionTransformer

## ▶️ How to Run
1. Clone this repository.
2. Install the required libraries.
3. Place the `train.csv` file in the project folder.
4. Open `funTransformer.ipynb` in Jupyter Notebook.
5. Run all cells sequentially.

## 📈 Learning Outcome
This notebook helps understand how feature transformations can improve data quality and prepare datasets for machine learning models.
