# 📅 Working with Dates and Time using Pandas

A beginner-friendly Jupyter Notebook demonstrating how to work with **dates and time** in **Python (Pandas)**. This notebook covers common date-time operations used in data analysis, feature engineering, and machine learning.

---

## 📌 Topics Covered

### 📆 Date Features
- Extract Year
- Extract Month
- Extract Day
- Extract Week of the Year
- Extract Quarter
- Extract Semester

### ⏳ Date Calculations
- Calculate time elapsed between dates
- Find days, months, and years passed
- Perform date arithmetic

### ⏰ Time Features
- Extract Hour
- Extract Minute
- Extract Second

### ⌛ Time Difference
- Calculate the difference between two timestamps
- Convert time differences into useful formats

## ⚠️ Note

Some code from older tutorials may not work with the latest versions of **Pandas (2.x)** and **NumPy**.

Examples include:
- `dt.isocalendar()` compatibility issues in older Pandas versions.
- `np.timedelta64(1, 'M')` no longer supported in recent NumPy versions.

Updated alternatives have been used wherever required.
